# Overview

TodoGo is my training project, the purpose of which is to learn
with the Spring Framework.<br>
It's built on a stack:<br>
   - Spring Boot , <br>
   - Spring Data JPA,<br>
   - Spring Security 5,<br>
   - Thymeleaf 5,<br>
   - PostgreSQL,<br>
   - and UIKit 3 for FrontEnd.
